Setup Steps

1

Verify Docker

Terminal
$ docker --version
Docker version 24.0.2, build cb74dfc

If Docker works ‚Üí continue to next step

2

Pull LocalStack Image

This downloads the "fake AWS" environment that runs in a container.

docker pull localstack/localstack

Why this matters:

  • LocalStack runs as a container
  • Contains all AWS service mocks
  • One-time download (~1.5GB)
  • Offline-capable after download
3

Run LocalStack

docker run -d \
  --name localstack \
  -p 4566:4566 \
  localstack/localstack
-d
Run in background
--name
Container name
-p 4566:4566
Port mapping
Check Status
$ docker ps
CONTAINER ID   IMAGE               STATUS             PORTS
a1b2c3d4e5f6   localstack/localstack   Up 2 minutes   0.0.0.0:4566->4566/tcp
4

Install AWS CLI

AWS CLI lets us "talk" to AWS (or LocalStack).

Linux (Debian/Ubuntu)

sudo apt install awscli -y

macOS (Homebrew)

brew install awscli

Windows

Download MSI from:

aws.amazon.com/cli/
Verify Installation
$ aws --version
aws-cli/2.13.0 Python/3.11.4 Linux/6.2.0-32-generic
5

Configure AWS CLI

LocalStack does not verify real AWS keys. Use dummy credentials:

Configure AWS CLI
$ aws configure
AWS Access Key ID [None]: test
AWS Secret Access Key [None]: test
Default region name [None]: us-east-1
Default output format [None]: json

‚ö†Ô∏è Critical Security Note:

These credentials only work with LocalStack. Never use them with real AWS! Real AWS requires real credentials from IAM.

üéØ Key Concept: Endpoint Switching

Real AWS

aws s3 ls

Connects to actual AWS S3 (needs internet + paid account)

LocalStack

aws s3 ls --endpoint-url=http://localhost:4566

Connects to local container (free + offline)

‚ö°

Same command, different backend

Just add

--endpoint-url=http://localhost:4566

Hands-On: Create S3 Bucket (LOCAL, FREE)

Create Your First Bucket

aws s3 mb s3://my-bucket \ --endpoint-url=http://localhost:4566
Response
{
"Location": "/my-bucket"
}

List Your Buckets

aws s3 ls --endpoint-url=http://localhost:4566
Response
2023-10-01 14:30:00 my-bucket

Add This
