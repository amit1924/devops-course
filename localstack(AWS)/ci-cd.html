7

Create and Mount Bare Git Repository

On Host Machine:

mkdir ~/repo.git cd ~/repo.git git init --bare

Creates a bare repository that acts as remote

Back in Container:

git remote add origin /repo.git git config --global --add safe.directory /repo.git git push -u origin master

Configures remote and pushes initial code

üîß Technical Details:

  • --bare Repository without working directory
  • /repo.git Mounted volume path (like GitHub remote)
  • safe.directory Fixes Docker/Git ownership warnings
  • -u origin master Sets upstream branch for future pushes

üéØ Success Indicators:

  • Bare repo created ‚Üí /repo.git
  • Remote added successfully
  • Git permission issues resolved
  • Push is now successful

Mounted bare repo simulates GitHub/GitLab remote
