‚úÖ Step-2 ‚Äî RUN MongoDB Container (Proper Setup)

Run MongoDB with volume attachment + port mapping

Command Breakdown:

  • -d ‚Üí run in background
  • --name mongo-db ‚Üí friendly name
  • -p 27017:27017 ‚Üí MongoDB port
  • -v mongoStore:/data/db ‚Üí persistent storage
  • mongo ‚Üí official MongoDB image
