Command:

This is:

Default scan (if you have root) Fast ‚ö° Reliable Slightly stealthy Most widely used

How it works:

  • Sends SYN packet
  • If reply = SYN/ACK ‚Üí port OPEN
  • If reply = RST ‚Üí port CLOSED
  • If no reply / ICMP blocked ‚Üí FILTERED

  • Less noise in logs

NULL / FIN / XMAS Stealth Scans

Null Scan:

nmap -sN <target>

Fin Scan:

nmap -sF <target>

Xmas Scan:

nmap -sX <target>

These are tricky stealth scans. They exploit weird TCP RFC behavior.

Logic:

  • No SYN packet
  • If port CLOSED ‚Üí sends RST

So results:

  • RST = CLOSED

  • ‚úîÔ∏è Bypass some old firewalls
  • ‚úîÔ∏è Sometimes stealthy

Problems:

  • ‚ùå Doesn't work properly on Windows
  • ‚ùå Many modern systems block these

Use when:

  • ‚úîÔ∏è Testing older Unix networks
  • ‚úîÔ∏è Need stealth alternatives
