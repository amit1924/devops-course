üéÆ Choosing Which Scripts to Run

Run a specific category

Example: vulnerability scripts

nmap --script vuln <target>

Discovery:

nmap --script discovery <target>

Run all HTTP-related scripts

nmap --script "http-*" <target>

Combine categories

nmap --script "default,safe" <target>

Boolean Logic (power move üòé)

nmap --script "(default or intrusive) and not http-*" <target>

Run intrusive + default but ignore HTTP scripts

üß© Script Arguments

Some scripts need parameters

Example:

nmap --script ftp-anon --script-args ftp-anon.maxlist=200 <target>

Example advanced format:

--script-args 'user=admin,pass=1234'

You can even load args from file:

--script-args-file file.txt

Viewing Script Info

Want to see what a script does?

nmap --script-help ftp-anon

Or see all vuln scripts descriptions:

nmap --script-help vuln

Debug NSE Traffic

Want to see EVERYTHING script sends/receives?

--script-trace

So cool for learning üëå
