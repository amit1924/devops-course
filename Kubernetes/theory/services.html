üß† What Is EndpointSlice?

Behind the scenes, when you create Service:

Kubernetes creates EndpointSlice object

EndpointSlice Contains:

List of healthy Pod IPs

Ready status

Ports

Node location

Example EndpointSlice data:

- addresses:
  - "10.1.0.5"
  - "10.1.0.6"
  - "10.1.0.7"

- name: http
  port: 3000
  protocol: TCP

When Pod dies:

Removed from EndpointSlice

When Pod created:

Added to EndpointSlice

Service forwards traffic to Pods listed in EndpointSlice

‚ùå Nobody does this.
