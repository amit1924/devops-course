üß† Very Important Concept

Kubernetes does NOT come with built-in Ingress controller.

You must install one.

The controller watches Ingress resources and configures itself automatically.

üåê Where Ingress Operates (Layer 7 Magic)

Ingress works at:

OSI Layer 7

(Application layer)

That means:

  • It understands HTTP
  • It can inspect Host header
  • It can inspect Path
  • It can inspect URL
  • It can handle TLS

Services cannot do this.

OSI Layer 4

(TCP only)

Services just forward packets. They don't understand HTTP.
